## Método DRE

Criamos o método DRE para tornar o aprendizado mais direto e aplicável:

- **D — Definição:** conceito explicado de forma clara, do fundamento ao avançado
- **R — Relevância:** conexão com a realidade de trabalho — carteiras, produtos, decisões estratégicas
- **E — Exemplo:** aplicação em situações reais do mercado, não apenas em questões de prova
